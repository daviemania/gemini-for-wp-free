# Gemini Project

This is the main project repository for the Gemini AI Assistant and related tools.

## Project Structure

- `agents/`: Contains various AI agents, including the Gemini AI Assistant.
- `docs/`: Documentation and templates.
- `gemini-mcp-tools/`: Custom WordPress plugin for Media Control Protocol (MCP) integration.
- `projects/`: Specific project-related files and documentation.
- `setups/`: Setup guides and configurations (e.g., VS Code, WordPress development).
- `smart-folder-manager/`: Node.js script for organizing files.
- `tasks/`: Task-specific files and notes.
- `vendor/`: Third-party dependencies.

## Development Workflow

1.  **Local Development:** WSL2 + Bitnami stack.
2.  **Version Control:** Git.
3.  **Code Editor:** VS Code with WSL integration.
4.  **Testing:** Staging environment on the same stack.
5.  **Deployment:** Manual (considering automation).

Refer to the `setups/` directory for detailed environment setup guides.

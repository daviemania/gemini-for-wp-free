# Gemini Project

This project is configured to interact with a WordPress site using a set of custom MCP functions.

## WordPress Integration

A comprehensive set of 37 MCP functions are available for managing the connected WordPress site. These tools provide capabilities for managing:

- Posts and Pages
- Users
- Comments
- Media Library
- Taxonomies (Categories and Tags)
- Site Options

## Available Tooling

### WordPress MCP Toolset

This project utilizes a powerful set of 37 MCP functions for comprehensive WordPress site management.

For a complete list of available functions, their parameters, and usage details, please see the [WordPress MCP Functions Documentation](wordpress_mcp_functions.md).

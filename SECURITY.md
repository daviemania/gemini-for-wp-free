# Security Policy

## Supported Versions

As the plugin is currently in its initial release phase (version 1.0.0), security support is focused solely on this version and future patches.

| Version | Supported |
| ------- | :---: |
| 1.0.0 | :white_check_mark: |
| < 1.0.0 | :x: |
